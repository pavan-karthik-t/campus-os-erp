# CampusOS Local Setup Guide

## Prerequisites

- Node.js 20+
- npm 10+

## 1. Install

```bash
npm install
```

## 2. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Demo Login

Use any of these local demo accounts:

| Role | Email | Password |
|------|-------|----------|
| Administrator | admin@campusos.local | password123 |
| Faculty | faculty@campusos.local | password123 |
| Student | student@campusos.local | password123 |

No external database, storage, or email service is required.

## Tests

npm test
